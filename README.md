# AI タスク依頼ワークショップ

## 概要

AI にタスクを依頼する練習用のバグあり React アプリです。

## アプリ一覧

| フォルダ       | 内容                    | バグ数 |
| -------------- | ----------------------- | ------ |
| `todo-app/`    | 入門編：Todo アプリ     | 1 つ   |
| `ec-cart-app/` | 応用編：EC カートアプリ | 3 つ   |

## 必要なもの

- Node.js 18 以上
- npm

## 起動方法

### 入門編（Todo アプリ）

```bash
cd work01
npm install
npm run dev
```

### 応用編（EC カートアプリ）

```bash
cd work02
npm install
npm run dev
```

### 上級編（ユーザー管理画面）

```bash
cd work03
npm install
npm run dev
```

いずれも http://localhost:5173 で起動します。
