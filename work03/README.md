# ユーザー管理画面（上級編）

AIタスク依頼ワークショップの上級編で使用するユーザー管理アプリです。

## バグ

このアプリには**5〜6個のバグ**があります。

- パフォーマンス問題
- 非同期処理の問題
- 状態管理の問題
- エラーハンドリングの問題
- UX上の問題

## ファイル構成

```
src/
├── App.jsx           # メインコンポーネント
├── SearchBar.jsx     # 検索バー
├── UserList.jsx      # ユーザー一覧
├── UserDetail.jsx    # ユーザー詳細/編集モーダル
├── api.js            # APIモック
├── hooks/
│   └── useUsers.js   # カスタムフック
├── main.jsx
└── index.css
```

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

## 機能

- ユーザー一覧表示
- 名前/メールで検索
- ユーザー情報の編集
- ユーザーの削除

## 注意

APIはモックで、10%の確率でランダムにエラーを返します。
