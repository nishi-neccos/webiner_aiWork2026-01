# ECカートアプリ（応用編）

AIタスク依頼ワークショップの応用編で使用するECカートアプリです。

## バグ

このアプリには**3つのバグ**があります。

- カートの小計がおかしい
- クーポンを入れても安くならない
- 在庫0の商品が買えてしまう

## ファイル構成

```
src/
├── App.jsx          # メインコンポーネント
├── ProductList.jsx  # 商品一覧
├── Cart.jsx         # カート表示
├── utils.js         # 計算ロジック
├── main.jsx
└── index.css
```

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

## クーポンコード

| コード | 割引率 |
|--------|--------|
| SAVE10 | 10%OFF |
| SAVE20 | 20%OFF |
| HALF   | 50%OFF |
